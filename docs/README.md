# MazeSpace Arena 3D

A fully configurable 3D Maze application with customizable themes, agents, and behaviors.

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run with defaults
python ex.py
```

## Features

-   **Configurable**: Customize every aspect via YAML or CLI.
-   **Themes**: Switch between Space, Forest, Lava, etc.
-   **Persistence**: Auto-saves your last session settings.
-   **Menu System**: Stack-based navigation with "Back" support.
